# 🛍️ React Product Listing UI

A modern and responsive **Product Listing Interface** built using **React + Vite**, showcasing products in a clean e-commerce layout with smooth loading experience and polished UI.

---

## 🚀 Live Demo

🔗 **Deployed Link:**
https://react-product-listing.vercel.app/

📦 **GitHub Repository:**
https://github.com/PushkarAdhikari/react-product-listing

---

## 📌 Features

* 🛍️ Clean and modern product grid layout
* ⚡ Fast performance using Vite
* 🌐 API integration (FreeAPI Products)
* 💀 Skeleton loading UI (shimmer effect)
* 🎨 Beautiful card design with hover effects
* ⭐ Styled rating badge with improved UI
* 📱 Fully responsive layout

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** CSS (custom modern UI)
* **API:** FreeAPI Products API
* **Deployment:** Vercel

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/PushkarAdhikari/react-product-listing.git
cd react-product-listing
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
